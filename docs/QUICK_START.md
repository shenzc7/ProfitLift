# Quick Start Guide

**Time to value: < 5 minutes**

1. **Install**: Run `ProfitLift-Setup.exe`.
2. **Launch**: Open the app.
3. **Get Data**: Download the [Sample CSV](../data/sample/sample_1k.csv) (or use your own).
4. **Upload**: Go to the **Upload** tab and drop the file.
5. **Analyze**: Go to **Bundles** and pick the top 3 recommendations.
6. **Act**: Send the list to your merchandising team!
